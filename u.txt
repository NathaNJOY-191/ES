### *How to Draw*

1. ATM System boundary box.

# ✅ *3. Sequence Diagram (Withdraw Cash Example)*

### *Actors*

* Customer
* ATM
* Screen / Keypad
* Bank Server
* Cash Dispenser

---

### ✔ *Sequence Flow*

1. Customer inserts card.
2. ATM requests PIN.
3. Customer enters PIN.
4. ATM sends PIN/Card details to Bank Server.
5. Bank Server validates user.
6. ATM asks customer for withdrawal amount.
7. Customer enters amount.
8. ATM requests bank server to check balance.
9. Bank server sends approval.
10. ATM activates Cash Dispenser.
11. Cash is dispensed.
12. ATM prints receipt.
13. ATM ejects card.

---

### ✔ *Sequence Diagram (Text Representation)*


Customer        ATM        Screen/Keypad     BankServer        CashDispenser
   |             |               |                |                   |
   |--InsertCard->               |                |                   |

   |             |<-enterPIN---- |                |                   |
   |             |--sendPIN---------------------->|                   |
   |             |<-------PIN OK------------------|                   |
   |             |--askAmount-->|                 |                   |
   |             |<-enterAmount-|                 |                   |
   |             |--checkBalance----------------->|                   |
   |             |<-------Approved-----------------|                   |
